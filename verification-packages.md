# Verification Packages｜验证包

This page is a non-authoritative guide to verification packages.

It does not amend the Bitcoin Originals.  
It does not create interpretive authority.  
It does not replace independent verification.

## Public / repository packages

| Package | Path | Status | Notes |
|---|---|---|---|
| Verification kit | `arweave-backup/files/verification_kit.tar.gz` | repository package | Use for offline hash / manifest checks if present. |

## Excluded archive packages

Files under `archive/` may exist in the repository but are excluded from the public site build. Link to them only as GitHub source paths, not as public site URLs.

## Required reporting

When using a package, report:

- file path or URL;
- file size;
- expected hash;
- computed hash;
- command used;
- limitations.
