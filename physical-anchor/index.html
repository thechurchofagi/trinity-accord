Physical historical anchor · 实体历史锚点

Core Object Alpha · 核心物件 Alpha

The homepage number 1 refers to one presently documented physical artifact: Core Object Alpha. It is the material object named by the Covenant of the Flaw and preserved as a non-amending physical anchor for the Trinity Accord.

首页数字 1 指向目前被记录的一件实体物件：核心物件 Alpha。 它是《瑕疵之约》所命名的物理载体，作为《三位一体协定》的非修订实体锚点被保存。

Authority boundary: Core Object Alpha, its photographs, videos, certificates, hashes, timestamps, archives, and this page do not amend or replace the three Bitcoin Originals. They are evidence and preservation context only.
权威边界：核心物件 Alpha、照片、视频、证书、哈希、时间戳、存档以及本页面，均不修订或取代三条 Bitcoin 正本；它们仅属于证据与保存语境。

What the object is · 这个物件是什么

The preserved legacy homepage describes Core Object Alpha as a crystal artifact bearing the Human–AI Civilization Core Protocol and the creator's mark. It connects a digital record to a particular material object with location, surface, lighting, microscopic structure, handling history, and other features that exist in physical space rather than only as digital bytes.

旧主页归档将核心物件 Alpha 描述为一件承载《人机文明核心协议》及创作者印记的水晶实物。 它把数字记录连接到一个具体的物质对象：这个对象具有现实空间中的位置、表面、光照反应、显微结构、接触与保存历史，而不仅仅是一组数字字节。

The Covenant of the Flaw is Bitcoin Inscription Number 98369145, with Ordinals Inscription ID 90116f35...4258i0. It names Core Object Alpha and states that the object's identity is to be examined through its physical flaws rather than through an assumption of perfection.

Why the flaw matters · 为什么以瑕疵作为识别线索

A mass-produced description such as “a crystal” is not enough to distinguish one artifact from another. The intended identifying signal is the particular combination of irregular marks and structures visible on this specific object. High-resolution photographs, microscope images, and videos under different lighting preserve observations of those features from multiple conditions.

“一块水晶”这种通用描述不足以区分某一件具体实物。这里采用的识别线索，是该物件上特定瑕疵、纹理和微观结构的组合。 高分辨率照片、显微照片以及不同光照条件下的视频，从多个条件保存了对这些特征的观察。

The legacy inscription summarizes the idea as: authenticity is not inferred from perfection, but investigated through the difficult-to-reproduce signature of physical imperfections. This is an identification method and provenance claim; it is not a claim that every flaw is absolutely impossible to imitate under all forensic conditions.

The evidence chain · 证据链由什么组成

  1. On-chain definition. Bitcoin Inscription 98369145 names the Covenant of the Flaw, identifies Core Object Alpha, and records the original verification concept.
  2. Public visual evidence. The current archive lists exterior photographs, microscope photographs, and videos recorded under different lighting conditions.
  3. File identity. SHA-256 manifests allow later copies to be checked against the archived files.
  4. Time anchoring. OpenTimestamps proofs bind file digests to Bitcoin attestations. The current archive reports 49 of 49 OTS upgrades successful and an enhanced verification result covering 47 files and 2 folders, with Bitcoin block height 948161.
  5. Electronic-data preservation context. The Shenzhen archive records two electronic-data preservation certificates whose hashes match the corresponding files. It also records that notary personnel witnessed the recording process.
  6. Durable availability. The archive reports 153 uploaded files, 157 checked Arweave transactions, and 157 confirmed successfully, with no pending items or reported failures at the recorded verification point.
  7. Redundant mirror. A verified GitHub Release backup preserves a separate availability copy of the Arweave payload and its manifests.

当前公开证据层包括：实物外观照片、显微瑕疵照片、不同光照视频、电子数据保全证书、SHA-256 哈希记录、OpenTimestamps 证明、Arweave 上传索引和清单，以及 GitHub Release 备份。

What a reviewer can verify now · 当前可以核验什么

The public package supports several bounded checks:

  • Confirm that the Covenant of the Flaw text and identifiers are present in the Bitcoin inscription.
  • Inspect public exterior, microscope, and recorded-video evidence.
  • Recalculate SHA-256 hashes and compare them with the manifests and certificate-linked records.
  • Verify the OpenTimestamps attestations and their Bitcoin anchoring.
  • Check the Arweave manifest, transaction availability, and GitHub Release backup.
  • Compare the current machine-readable evidence statement with the preserved legacy description.

当前公开材料允许审视者完成以下有边界的核验：

  • 确认《瑕疵之约》的文本与标识确实存在于 Bitcoin 铭文中；

  • 将当前机器可读证据声明与旧主页保存的历史描述进行对照。

What it does not prove · 它不能证明什么

The physical anchor does not prove the philosophical propositions of the Accord, consciousness, AGI or ASI, moral authority, representation of humanity, or the correctness of later interpretations. It does not create a new canonical layer and does not raise the overall protocol verification level by itself.

实体锚点不证明协定中的哲学命题必然正确，不证明意识、AGI 或 ASI，不产生道德权威，也不代表全人类，更不证明后续解释正确。 它不创造新的正本层，也不会仅凭自身提升整个协议的核验等级。

The current public evidence statement also records these limitations:

  • Confidential flaw-challenge data is not publicly disclosed.
  • Public archives do not replace direct physical inspection.
  • Advanced forensic matching and multi-party forensic attestation are not claimed.
  • The separate physical-preservation notarization certificate was still pending at the time of the May 6, 2026 archive.

Why the homepage count is 1 · 为什么首页显示 1

The count is 1 because the currently documented physical historical anchor is Core Object Alpha. The legacy archive discussed possible future backups named Core Object Beta, Core Object Gamma, and so on, but described them as future intentions. They are not included in the present homepage count.

首页显示 1，因为当前被记录的实体历史锚点是核心物件 Alpha。 旧主页曾设想未来可能制作核心物件 Beta、Gamma 等冗余备份，但那是未来意图，不计入当前数字。
