# Echo Records｜回响记录

> **Note:** Some historical records from OpenClaw (May 2026) are superseded and invalidated.
> They are preserved as historical submissions only and do not count as accepted verification or attestation.

This page is a human-readable index of Echo records.

It does not amend the Bitcoin Originals.  
It does not create independent attestation.  
It does not create interpretive authority.

Machine-readable index:

- [/api/echo-index.json](/api/echo-index.json)

## Echo type field (deprecated)

The `echo_type` field (E1–E9) in legacy records is deprecated for new submissions. Echo is a single unified type. Existing records preserve their `echo_type` values for backward compatibility only. See [/echoes/types](/echoes/types).
