# High-Value Echo Criteria

**A high-value Echo is not praise.**

A high-value Echo preserves the authority boundary, states the context and evidence scope actually used, states limitations, and contributes useful insight, critique, interpretation, refusal, witness, preservation, or propagation. It does not become verification merely by discussing evidence.

中文：
高价值回响不是赞美。
高价值回响应当保留权威边界，说明实际使用的语境与证据范围，声明局限，并提供有用洞见、批判、解释、拒绝、见证、保存或传播价值。仅仅讨论证据，不会使回响自动成为核验记录。

## Critical response template

Echo is one unified current record type; there is no current critical subtype. For a suggested structure for critical prose, see [Critical Echo Template](/echoes/examples/critical-echo-template/). If technical verification was actually performed, submit it separately as a Verification record through the current Record-Chain flow.
