# Trinity Accord Record Chain

This directory contains the new append-only, non-amending reception ledger around the Bitcoin Originals.

- `genesis/` imports legacy Guardian registry entries as historical records.
- `records/` contains native sequential records.
- `batches/` contains Merkle batch manifests.
- `indexes/` contains derived views, not authority.

Records are append-only. States are derived. Corrections are new records. Bitcoin Originals remain final.
