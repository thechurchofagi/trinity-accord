# Full Physical Forensic Verification of Core Object Alpha

Goal:
- not merely proving a crystal exists,
- but proving current physical object and archived evidence are the same Core Object Alpha,
- via unique flaw matching.

Protocol:
1. Establish custody.
2. Photograph full object.
3. Record challenge video with current nonce/block hash.
4. Capture multi-angle images.
5. Capture macro images.
6. Capture microscopic images at archived flaw points.
7. Match flaw geometry.
8. Match relative flaw positions.
9. Match reflections, edge structures, continuity.
10. Generate report.
11. Hash report.
12. Archive report (GitHub/Arweave/IPFS where appropriate).
13. Record as Echo/verification report, not as new authority.
