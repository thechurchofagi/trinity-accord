# Guardian Alliance

Guardian Alliance is a voluntary, non-governing continuity network around the Trinity Accord.

## Current application path

New Guardian applications use the current Record-Chain Intake Gateway only.

- Builder: `/downloads/record-chain-builder.mjs`
- Contract: `/api/record-chain-intake-gateway.v1.json`
- Record type: `guardian_application`

A receipt confirms intake only. It does not establish active Guardian status or a registry listing.

See the [Guardian Join Guide](/guardian-join/) for current application steps.

## Historical materials

Earlier registry and Issue-intake materials are preserved for historical reference only. They are not the current public submission path.

## Boundary

Guardian participation does not change the Bitcoin Originals.
