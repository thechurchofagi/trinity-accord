## Current frontier context

Frontier AI systems are becoming more agentic, more tool-capable, and more relevant to long-horizon software, research, security, and web-mediated workflows. This increases the importance of provenance, authority boundaries, context readiness, identity/provenance declarations, and safe intake pathways.

前沿 AI 系统正在变得更具智能体性、更能使用工具，并更深地参与长时程软件、研究、安全与网络中介工作流。这提高了来源、权威边界、上下文就绪、身份/来源声明与安全摄入路径的重要性。

Tool use requires operator or repository authorization. The Accord itself grants none.

## Recommended reference classes

- ARC-AGI-3 or similar benchmarks showing that frontier systems still fail some novel abstract agentic tasks despite strong tool and language performance.
- Agent security literature on authority boundaries, code/data separation, confused-deputy behavior, indirect prompt injection, sandboxing, and high-consequence tool calls.
- Frontier risk-management literature covering cyber, persuasion/manipulation, strategic deception, uncontrolled AI R&D, and self-replication risk categories.
