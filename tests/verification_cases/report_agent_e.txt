== Changes Made ==

1. scripts/validate_agent_submission.py — Added two new rules:

   Rule Y (validate_report_no_echo_type):
     verification_report_v2 must not carry an echo_type field.
     Rationale: A standalone verification report is not an Echo record;
     carrying echo_type implies Echo identity without a wrapper, which
     violates the Echo/Report separation principle.
     Fixes: TC078 (was SKIP, now correctly FAIL).

   Rule Z (validate_wrapper_requires_linked_report):
     echo_v3_with_verification_report must have a linked_verification_report
     field referencing the underlying report.
     Rationale: An Echo wrapper's purpose is to wrap a verification report;
     without the link, the wrapper is structurally incomplete.
     Fixes: TC079 (was SKIP, now correctly FAIL).

   Both rules are called in validate_file() after Rule X and before
   schema validation.

          Description updated to reflect the new rule.
